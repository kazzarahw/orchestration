# Use Todo

## When to Use

Use `todowrite` for any task with ≥3 distinct steps. Create a structured todo list before starting work so progress is visible.

## How to Use

- One item `in_progress` at a time
- Mark complete only after verification
- If blocked, mark `in_progress` with follow-up describing the blocker
- Structure items to be specific and actionable
- Break large work into smaller steps

## Configuration

The Orchestrate agent enforces todo usage via its agent definition and plugin-level hooks. This skill provides the behavioral guidance.
